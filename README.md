# DataMergeAI

Projeto para processamento inteligente de grandes planilhas Excel usando IA.

## Estrutura
- **backend/**: API e processamento com Django
- **frontend/**: Interface do usuário com React (Vite)

## Objetivo
- Filtrar, limpar e otimizar dados de planilhas Excel
- Permitir upload de múltiplas planilhas
- Exportar dados limpos para Excel e enviar por e-mail

## Como rodar
Veja instruções nas pastas `backend` e `frontend` para rodar cada parte do projeto.
